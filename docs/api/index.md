# API Reference

This page provides an overview of the `torch-pharma` API.

## Modules

### Molecules
- `torch_pharma.molecules.Molecule`: Core molecule class for handling chemical structures.

### Models
- `torch_pharma.models.gnn.GCN`: Graph Convolutional Network.

### Tasks
- `torch_pharma.tasks.PropertyPredictionTask`: Task for predicting molecular properties.

### Training
- `torch_pharma.training.Trainer`: Unified trainer for models and tasks.
